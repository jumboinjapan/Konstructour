## 📏 Размеры и отступы

```
Высота карточки: авто (зависит от контента)
Ширина: 100% контейнера

Padding основной области: 20px (p-5)
Gap между элементами: 12px (gap-3)
Border radius: 8px (rounded-lg)
Border width: 2px (border-2)
Shadow: sm → lg (при hover)

Иконка категории:
  Размер: 48x48px (w-12 h-12)
  Border radius: 8px (rounded-lg)
  
Заголовок:
  Font size: 18px (text-lg)
  Font weight: 600 (font-semibold)
  
Подзаголовок:
  Font size: 14px (text-sm)
  Color: gray-500
```
